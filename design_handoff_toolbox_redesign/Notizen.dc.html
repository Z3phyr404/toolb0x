Deployment-Setup

Das Portal läuft auf einem Hetzner VPS mit Node.js 20 und PostgreSQL 15. Alle sensiblen Daten werden clientseitig mit AES-256-GCM verschlüsselt.

Wichtige Befehle

npm run db:migrate
npm run db:seed
npm run dev
  • JWT-Secret per crypto.randomBytes(64) generieren
  • DATABASE_URL in .env anpassen
  • Portal unter /portal öffnen
Selbst der Betreiber kann die verschlüsselten Nutzerdaten nicht lesen — Zero-Knowledge by design.
