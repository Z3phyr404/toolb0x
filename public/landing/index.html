// SICHERHEIT

Zero-Knowledge, ernst gemeint

Deine Daten werden auf dem Server verschlüsselt gespeichert — der Schlüssel entsteht aus deinem Passwort und existiert nur, während du angemeldet bist.

Verschlüsselt, bevor gespeichert wird

Jeder Eintrag wird mit AES-256-GCM verschlüsselt, bevor er die Datenbank erreicht — Namen, Beträge, Notizen, Passwörter.

Der Server ist blind

In der Datenbank steht nur unlesbarer Chiffretext. Selbst mit vollem Serverzugriff lässt sich nichts entziffern.

Recovery-Code als Rettungsanker
